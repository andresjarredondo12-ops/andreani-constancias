Conversor Masivo HTML → PDF

Selecciona múltiples archivos HTML o una carpeta completa. El sistema abrirá cada archivo y permitirá imprimirlo en PDF.

Importante:
Los navegadores NO permiten generar PDFs automáticamente sin interacción del usuario por razones de seguridad.

Este sistema:
  • Carga múltiples archivos HTML.
  • Los procesa uno por uno.
  • Abre el diálogo real de impresión del navegador.
  • Permite guardar cada uno como PDF.
